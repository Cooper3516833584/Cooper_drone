# No-props bench test checklist

- Mechanical propellers are removed.
- Battery is fixed securely.
- Flight controller power is stable.
- Companion computer power is stable.
- Serial wiring is secure.
- Heartbeat can be received.
- Armed state and mode can be read.
- Do not send takeoff.
- Test only healthcheck and safety state.
- Test kill switch behavior.
- Test manual mode takeover behavior.
